# Micros-Batiz
En el presente se contienen algunas prácticas correspondientes a las materias de Arquitectura de Microcontroladores y Microprocesadores, impartidas en el CECyT 9 "Juan de Dios Bátiz"
correspondiente al plan de estudios 2009 de la carrera de "Técnico en Sistemas Digitales".

Los programas y archivos contenidos en el repositorio están desarrollados para el microcontrolador PIC16F877A de Microchip.
Para el desarrollo de los programas en lenguaje ensamblador se empleó el software de MPLAB IDE y para las simulaciones el software Proteus 8.

Las prácticas con el formato "x.6" son aquellas que pertenecen/pertenecían al 6to semestre de la carrera técnica, aquellas que no, pertenecen/pertenecían al 5to
semestre de la carrera.

Para visualizar mejor los códigos en la página web de Github se recomienda no traducir la página.

El archivo denominado "Machote.ASM" es una plantilla de código que puede ser usada para desarrollar código de manera estructurada.
